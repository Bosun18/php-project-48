### About the program
#### Вычислитель отличий – программа, определяющая разницу между двумя структурами данных.
### Requirements:
#### PHP >= 8.0
#### Composer >= 2.5
#### Make >= 4.3
### Install:
#### 1. `git clone git@github.com:Bosun18/php-project-48.git`
#### 2. `cd php-project-48`
#### 3. `make install`
